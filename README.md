# MSP Growth Platform

A comprehensive business intelligence platform for Managed Service Providers.

## Features

- Client Management & Analytics
- Financial Performance Tracking
- Software License Optimization
- Sales Pipeline Management
- AI-Powered Insights
- Custom Reporting
- Third-party Integrations

## Tech Stack

- **Frontend**: React, Chart.js, CSS3
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JWT, bcrypt
- **Security**: Helmet, CORS, Rate Limiting

## Setup Instructions

### Prerequisites
- Node.js 16+
- MongoDB
- npm or yarn
